## More Labs
Now we will create another Dynamic Web Application to display a randomly generated name.

1. Create a Dynamic Web Project called `NameChooser`.

1. Create a servlet that returns an HTML document containing a randomly chosen first and last name. Sample names are listed below.

1. Map the servlet to a URL in your `web.xml`, and set the welcome page to that URL.
