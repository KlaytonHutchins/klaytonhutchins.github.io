## Web Data and Dynamic Applications

We can send data to our Web Applications. In this chapter, we will learn how.

### Objectives

* Understand how HTML `<form>` data arrives at a Servlet.

* Understand the difference between `GET` and `POST` data.

* Be able to use data from `HttpServletRequest` in a Servlet.
