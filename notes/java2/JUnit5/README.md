The release of JUnit version 5 included many features and changes, which we will highlight in this section.

### Objectives
* Be able to use `@BeforeEach` and `@AfterEach` to set up test cases.
* Be able to use `@DisplayName` and other new JUnit 5 annotations.
* Be able to use lambdas and `Executable` to test Exceptions.
